# Finalizar trabalho na issue atual

Finalize o trabalho na issue do Linear `$ARGUMENTS`.

1. Rode os testes relevantes:
   - Se houve mudanças em `api/`: `cd api && npm run test`
   - Se houve mudanças em `app/`: `cd app && npm run test`
2. Rode o lint:
   - Se houve mudanças em `api/`: `cd api && npm run lint`
   - Se houve mudanças em `app/`: `cd app && npm run lint`
3. Se testes e lint passaram, faça o commit seguindo as convenções (emoji + português)
4. Atualize a issue no Linear para **In Review**
5. Adicione um comentário na issue do Linear resumindo o que foi feito

Se testes ou lint falharem, corrija os problemas antes de continuar.
